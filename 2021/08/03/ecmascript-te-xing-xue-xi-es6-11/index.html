ECMAScript特性学习es6-11

ES全称ECMAScript，ECMAScript是ECMA制定的标准化脚本语言。目前JavaScript使用的ECMAScript版本为ECMAScript-262。

ECMAScript 标准建立在一些原有的技术上，最为著名的是 JavaScript (网景) 和 JScript (微软)。它最初由网景的 Brendan Eich 发明，第一次出现是在网景的 Navigator 2.0 浏览器上。Netscape 2.0 以及微软 Internet Explorer 3.0 后序的所有浏览器上都有它的身影。

es发展至今已有7个版本，在此主要讨论6~11。后续新出则另写

ECMAScript版本 发布时间 新增特性
ECMAScript2009（ES5） - 扩展Object、Array、Function功能等
ECMAScript2015(ES6) - 类、模块化、箭头函数、函数参数默认值等
ECMAScript2016(ES7) - includes，指数操作符
ECMAScript2017(ES8) - sync/await，Object.values(),Object.entries(),String padding等
ECMAScript2018(ES9) -
ECMAScript2019(ES10) -
ECMAScript2020(ES11) -

ES6 新特性

类（Class）

ES6提供了更接近传统语言的写法，引入Class的概念，作为对象的模板。通过class关键字，可以定义类。基本上，ES6的class可以看作只是一个语法糖，它的绝大部分功能，ES5都可以做到，新的class写法只是让对象原型的写法更加清晰、更像面向对象编程的语法而已

通过es5和es6的两种写法看区别

初识

//es5
function Phone(brand,price){
	this.brand = brand
	this.price = price
}
//添加方法
Phone.prototype.call = function(){
    	console.log("打电话")
}

let phone = new Phone("XiaoMi",2999)
phone.call();
console.log(phone);
//es6
class Phone{
	//构造方法固定写法
	constructor(brand,price){
		this.brand = brand
		this.price = price
	}
	call(){
		console.log("打电话")
	}
}
let phone = new Phone("Leven",999)

类继承

//es5
function Phone(brand,price){
    this.brand = brand
    this.price = price
}
Phone.prototype.call = function(){console.log("打电话")}

//继承
function SmartPhone(brand,price,color,size){
    Phone.call(this,brand,price)
    this.color = color
    this.size = size
}

SmartPhone.prototype = new Phone
SmartPhone.prototype.constructor = SmartPhone
//es6
class Phone{
    //构造方法
    constructor(brand,price){
        this.brand = brand
        this.price = price
    }
    
    //父类成员属性
    call(){
        console.log("打电话")
    }
}

//继承
class SmartPhone extends Phone{
    constructor(brand,price,color,size){
        super(brand,price)
        this.color = color
    	this.size = size
    }
}

模块化

模块化是指将一个大的程序文件，拆分成许多小的文件，然后将小文件组合起来。

优点：防止命名冲突；代码复用；高维护性

  • export：用于规定模块的对外接口
  • import：用于输入其他模块提供的功能

​ ES5不支持原生的模块化，在ES6中模块作为重要的组成部分被添加进来。模块的功能主要由 export 和 import 组成。每一个模块都有自己单独的作用域，模块之间的相互调用关系是通过 export 来规定模块对外暴露的接口，通过import来引用其它模块提供的接口。同时还为模块创造了命名空间，防止函数的命名冲突。

export（暴露）

test.js

//单个暴露
export let name = 'test'
export findJob(){console.log("找工作")}
//统一暴露
let name = "test"
findJob(){console.log("找工作")}
export{name,findJob}
//默认暴露
export default {
    let name = "test"
	findJob(){console.log("找工作")}	
}

import（引入）

//通用引入方式
<script type="module">
	import * as test from ./test.js
</script>
//解构赋值方式
<script type="module">
	import {name as userName,findJob} from ./test.js
	import {default as test} from ./test.js
</script>
//简写，仅针对默认暴露有效
<script type="module">
	import test from ./test.js
</script>
//通过新建一个js文件作为入口，在app.js中进行引入。在所需html中直接
<script src="./app.js" type="module"></script>

箭头函数

=>不只是关键字function的简写，它还带来了其它好处。箭头函数与包围它的代码共享同一个this,能帮你很好的解决this的指向问题。有经验的JavaScript开发者都熟悉诸如var self = this;或var that = this这种引用外围this的模式。但借助=>，就不需要这种模式了。

(a,b) => a+b
//相当于
function(a,b){return a+b}

// 箭头函数的例子
()=>1
v=>v+1
(a,b)=>a+b
()=>{
    alert("foo");
}
e=>{
    if (e == 0){
        return 0;
    }
    return 1000/e;
}

​ 不论是箭头函数还是bind，每次被执行都返回的是一个新的函数引用，因此如果你还需要函数的引用去做一些别的事情（譬如卸载监听器），那么你必须自己保存这个引用。

  • 箭头函数中的this指向windows。而function()中的this执行当前函数内
  • 箭头函数不能作为构造实例化对象
  • 箭头函数不能使用 arguments 变量

函数参数默认值

ES6支持在定义函数的时候为其设置默认值

//设置默认值
foo(height = 100, color = 'red'){
    
}

//不使用默认值
foo(height,color){
    let height = height || 60
    let color = color || 'yellow'
}

这样写一般没问题，但当参数的布尔值为false时，就会有问题了。比如，我们这样调用foo函数：

foo(0, "")

因为0的布尔值为false，这样height的取值将是50。同理color的取值为‘red’。

所以说，函数参数默认值不仅能是代码变得更加简洁而且能规避一些问题。

模板字符串

ES6支持模板字符串，使用字符串的拼接更加的简洁、直观

//不使用模板字符串
let max = '500'
let output = 'the max value = ' + max

//使用模板字符串
let max = 500
let output = `the max value ${max}`
  • 使用模板字符串时，将变量放在 ${} 中
  • 外面的不是引号，而是 ``

解构赋值

可以方便的从数组或对象中快速提取值赋给定义的变量

获取数组中的值

从数组中获取值并赋值到变量中，变量的顺序与数组中对象顺序对应

var num = ['one','two','three','four']
let [first,second,third,fourth] = num
console.log(first)  //one

//忽略某些值
var num = ['one',,,'four']
let [first,second,third,fourth] = num
console.log(first)  //one

//进一步简洁
let first,second,third,fourth

[first,second,third,fourth] = ['one','two','three','four']

//若未从数组中获取值，可以设置默认值
let a,b
[a = 5, b = 6] = [1]
console.log(a)   //1
console.log(b)   //6

//通过解构赋值可以方便交换两个变量的值
let a = 1
let b = 2
[a,b] = [b,a]
console.log(a)   //2
console.log(b)   //1

获取对象中的值

const user = {
    name: 'test',
    age: 18,
    city: 'ShangHai'
}

let {name,age,city} = user
console.log(name)   //test

延展操作符（Spread operator）

延展操作符 ...可以在函数调用/数据构造时，将数组表达式或者string在语法层面展开’还可以在构造对象时，将对象表达式按key-value的方式展开

  1. 数组的合并

    在没有展开语法时，只能通过push，splice，concat等方法来将已有数组元素变成新数组的一部分。有了展开语法则更简单。

    let user = ['Ben','Kitty']
let roomMate = ['Peter','Jim',...user]   //['Peter','Jim','Ben','Kitty'] 

//等同于
let person = ['Peter','Jim']
person.cancat(user)
    • ... 在狗仔参数列表和字数组时，可以在任意位置多次使用

  2. 数组克隆

    let user = ['Ben','Kitty']
let person = [...user]
person.pust('Jom')
console.log(person)   //['Ben','Kitty','Jom']
    • 展开语法和Object.assign()行为一致，执行的都是浅拷贝（只遍历一层）

  3. 在es9中新增对对象的支持

    var obj1 = { foo: 'bar', x: 42 };
var obj2 = { foo: 'baz', y: 13 };

var clonedObj = { ...obj1 };
// 克隆后的对象: { foo: "bar", x: 42 }

var mergedObj = { ...obj1, ...obj2 };
// 合并后的对象: { foo: "baz", x: 42, y: 13 }

  4. React中的应用

    通常在封装一个组件时，会对外公开一些props用于实现功能。大部分情况下在外部使用都应显示的传递props。但是当传递大量props时，会十分繁琐，此时可以使用...（用于取出参数对象所有可遍历属性）来进行传递

<div name="test" id="gid"></div>

//使用延展操作符
let prop = {
    id: 'gid',
    name: 'test'
}

<div ...prop></div>

  1. 配合解构赋值避免传递一些不需要的参数

    const user = {
    name: 'test',
    age: ‘18，
    sex: '男'
}
var {age,...other} = user

<div name="dixBox" number={2} {...other}>
    //等价于
<div name="dixBox" number={2} age='18' sex="男">

对象属性简写

在ES6中允许我们在设置一个对象的属性的时候不指定属性名

//不使用ES6,对象中必须包含属性和值，冗余
const user = {name: 'test',age: 12}

const user2 = {
    name: name,
    age: age
}
console.log(user2)   // {name: 'test',age: 12}
//使用ES6，可以直接写变量
const user = {name: 'ben',age: 18}
const user2 = {name,age}
console.log(user2)   // {name: 'test',age: 12}

Promise

Promise 是异步编程的一种解决方案，比传统的解决方案callback更加的优雅。

//不使用ES6  嵌套两个setTimeout回调函数
setTimeout(function(){
    console.log("一秒后输出hello")
    setTimeout(() => {
        console.log("两秒后输出hi")
    },1000)
},1000)
//使用ES6，在此处通过两个then来进行异步编程串行化，避免回调地狱
var waitFunction = new Promise(resolve,reject){
    setTimeout(resolve,1000);
}

waitFunction.then(function(){
    console.log("1秒后");
    return waitFunction
}).then(() => {
    console.log("2秒后")
})

支持let和const

在es6之前js是没有块级作用域的，const和let填补了这方面的空白，均为块级作用域

{
  var a = 10;
}

console.log(a); // 输出10

使用let与const定义的变量为块级作用域：

{
  let a = 10;
}

console.log(a); //-1 or Error“ReferenceError: a is not defined”

ES7新特性

includes()

  • includes() 函数用来判断一个数组是否包含一个指定的值。如果有返回true，否则返回false
  • indexOf() 函数验证数组中是否存在某个元素，只是需要根据返回值是否为-1 来进行判断
//indexOf()
let arr = ['react','angular','vue']
if(arr.indexOf('react') !== -1){
    console.log("react存在")
}

//includes()
let arr = ['react','angular','vue']
if(arr.includes('react')){
     console.log("react存在")
}

指数操作符

指数运算符**, 具有与Math.pow(..)等效的计算结果

  • 不使用指数操作符，则通过自定义的递归函数calculateExponent或者Math.pow()进行指数计算
function calculateExponent(base,exponent){
    if(exponent === 1){
        return base
    }else{
        return base * calculateExponent(base,exponent-1)
    }
}

console.log(calculateExponent(2,10)) //输出1024
console.log(Math.pow(2,10))   //输出1024
  • 使用指数函数**
console.log(2**10)   //输出1024

ES8新特性

async/await

在ES8中假如了对async/await支持，也就是“异步函数”

async/await详解

Object.values()

Object.values()是一个与Object.keys类似的函数，但返回的是Object自身属性的所有值，不包括继承的值

//通过Object.values() 遍历对象的所有值
const obj = {a:1,b：2,c:3}
const values = Object.keys(obj).map(key => {
    console.log(values)   //[1,2,3]
})

//使用Object.values()
const obj = {a:1,b：2,c:3}
console.log(obj.values()); //[1,2,3]

Object.entries

Object.entries()函数返回一个给定对象自身可枚举属性的键值对的数组

//遍历obj对象的所有属性的key和value,foreach
let obj = {name:"test",age:19}
Object.keys(obj).forEach(key => {
    console.log("key:"+key+'value:'+value)
})
//使用Object.entries()函数
for (let [key,value] of Object.entries(obj)){
    console.log(`key: ${key},value: ${value/}`)
}

padStart && padEnd

实例函数

  • String.padStart(targetLength,[padString])：允许将字符串或其他字符天机道原始字符串的开头
  • String.padEnd：允许将字符串或其他字符天机道原始字符串的结尾
  1. targetLength：当前字符串要填充到的 目标长度 。若数值小于当前字符串长度了，则返回当前字符串本身
  2. padString：（可选）填充字符串。若填充后的字符串的长度超过了目标长度，则只保留最左侧部分，缺省值为“”
console.log('0.0'.padStart(4,'10')) //10.0
console.log('0.0'.padStart(20))//                0.00

String.padEnd(targetLength,padString])

  • targetLength:当前字符串需要填充到的目标长度。如果这个数值小于当前字符串的长度，则返回当前字符串本身。
  • padString:(可选) 填充字符串。如果字符串太长，使填充后的字符串长度超过了目标长度，则只保留最左侧的部分，其他部分会被截断，此参数的缺省值为 “ “；
console.log('0.0'.padEnd(4,'0')) //0.00
console.log('0.0'.padEnd(10,'0'))//0.00000000

结尾允许参数列表使用逗号
